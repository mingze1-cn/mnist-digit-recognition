## 项目概述

本项目实现了一个完整的MNIST手写数字识别系统，使用TensorFlow框架和卷积神经网络(CNN)。系统包含两个主要部分：
1. **模型训练模块**：使用MNIST数据集训练一个高性能的CNN模型
2. **数字预测模块**：加载训练好的模型，对新的手写数字图像进行识别和分类

## 功能特点

- 使用三层卷积神经网络架构
- 包含数据预处理和归一化
- 自动图像大小调整和颜色反转
- 实时预测结果显示（含置信度）
- 支持批量图像处理
- 模型保存和加载功能
- 训练/验证精度差值计算
- 中文界面支持
